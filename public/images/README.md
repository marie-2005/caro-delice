# Images des Produits

Placez vos images dans ce dossier avec les noms suivants :

1. **crepe-nature.jpg** - Pour la Crêpe Nature (Image 1)
2. **crepe-nutella.jpg** - Pour la Crêpe au Nutella (Image 2)
3. **crepe-cerelac.jpg** - Pour la Crêpe au Cérélac (Image 3)
4. **crepe-jambon-fromage.jpg** - Pour la Crêpe au Jambon/Fromage (Image 4)
5. **crepe-viande-fromage.jpg** - Pour la Crêpe au Viande/Fromage (Image 5)

## Formats supportés
- JPG/JPEG (recommandé)
- PNG
- WebP

## Tailles recommandées
- Largeur : 400-800px
- Format : Paysage (ratio 4:3 ou 16:9)
- Poids : < 500KB par image pour de meilleures performances
